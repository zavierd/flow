# 📋 直接粘贴导入功能使用指南

## 🎯 功能概述

现在您可以直接将AI模型输出的表格数据粘贴到导入页面，无需保存为CSV文件！这大大简化了导入流程。

## 🚀 使用步骤

### 1. 准备数据
使用增强版AI提示词生成标准化的15列表格数据

### 2. 打开导入页面
访问：http://localhost/import/

### 3. 选择导入方式
- ✅ **模板类型**：选择"AI数据格式"
- ✅ **导入方式**：选择"📋 直接粘贴"

### 4. 粘贴数据
将AI模型输出的表格数据直接粘贴到文本框中

### 5. 开始导入
填写任务名称，点击"🚀 开始导入产品数据"

## 📊 支持的数据格式

### 1. **Markdown表格格式**（推荐）
```
| 产品描述 (Description) | 产品编码 (Code) | 系列 (Series) | ... |
| :--- | :--- | :--- | :--- |
| 单门底柜<br>1 door base unit | N-U30-7256-L/R | N | ... |
```

### 2. **制表符分隔格式**（Excel复制）
```
产品描述 (Description)	产品编码 (Code)	系列 (Series)
单门底柜	N-U30-7256-L/R	N
```

### 3. **CSV格式**
```
产品描述 (Description),产品编码 (Code),系列 (Series)
单门底柜,N-U30-7256-L/R,N
```

## ✨ 智能处理功能

### 🔄 **自动格式识别**
- 自动识别Markdown表格、制表符分隔、CSV格式
- 智能转换为标准CSV格式

### 📝 **数据预处理**
- 自动处理`<br>`换行符
- 清理价格数据中的逗号分隔符
- 处理空值和占位符

### 🎯 **即时导入**
- 无需保存文件，直接处理
- 实时反馈导入结果
- 详细的错误信息
